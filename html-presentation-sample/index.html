Acceleo is an open source code generator implementing the OMG's MOFM2T standard that uses any kind of EMF based models (UML, SysML, DSM, etc.) to generate any kind of textual language (Java, Scala, DSL, etc.) while keeping the traceability of the generated text.

The history of the Acceleo project

  • 2005 Birth of Acceleo
  • 2006 Acceleo 1.0
  • 2008 Entrance in the Eclipse Foundation
  • 2010 Acceleo 3.0 in the release train
  • 2011 Acceleo 3.1 released with Indigo
